# Code of Conduct

## Our Standards

This is a security research project. Contributors are expected to:

- Use this tool only on systems they own or have explicit authorization to test
- Share findings responsibly — report vulnerabilities privately before disclosing publicly
- Be constructive in code review and issue discussions
- Respect that others may be at different experience levels

## Not Acceptable

- Using this project to attack systems without authorization
- Submitting code designed to harm users or bypass safety systems
- Sharing outputs from this tool as actual attack tools

## Enforcement

Violations can be reported to brian.thomas.t@gmail.com.
Malicious contributions will be reverted and the contributor blocked.
